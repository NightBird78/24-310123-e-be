1)
получить ссылку на желанный репозиторий, который скопировать.
зайти в робочую path где есть репозитории (или просто туда где нужно жтот репозиторий)
потом в нужном path прописать "git clone 'git@github.com:user/repo.git'"
2)
перейти в нужний перозиторий на локальном диске и прописать "git pull", что бы получить последние изменения
3)
git status    // показывает состояние текущего репозитория на локальном диске и есть ли у него изменения
git log       // показывает список сделанных коммитов
git remote -v // показывает что локальный репозиторий связан с удаленным репозиторием под именем "origin" и указывает URL-адреса
